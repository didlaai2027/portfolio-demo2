# Development Notes

## To customize this site:
1. Add your photo: `images/profile.jpg`
2. Add your resume: `resume/Isha_Didla_Resume.pdf`
3. Update content in `index.html` (name, experience, projects, etc.)
4. Adjust colors in `css/style.css` under `:root` variables

## To deploy on GitHub Pages:
1. Create a new GitHub repo (e.g., `isha-didla-portfolio`)
2. Push all files in `portfolio-demo2/` to the repo root
3. Go to Settings > Pages > Source: Deploy from branch `main`
4. Your site will be live at: `https://yourusername.github.io/isha-didla-portfolio`
